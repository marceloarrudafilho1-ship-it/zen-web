/* how it works */

Paste an address. Get the whole picture.

From "what wallet am I looking at?" to "where did the money go?" in under sixty seconds.

  1. 01

    Paste any address

    EVM hex, Solana base58, or an ENS / Lens / Farcaster handle. Zen auto-detects the chain, fetches every transfer, and prices it against history.

    ▌ vitalik.eth
  2. 02

    Read the room

    Balance over time, top in/out/swap flows, counterparty labels, temperature tag, cluster suggestions — all rendered in milliseconds.

  3. 03

    Trace and pivot

    Follow-the-Money chases hops automatically. Bridge Pivot drops you onto the destination chain. web3.bio resolves identities. Save the case as a single file.

    trace pivot resolve save

  • invite-only access

    Sign-up requires a code only the operator shares. Rotate the code with one Railway variable to revoke future signups.

  • encrypted at rest

    Passwords hashed with bcrypt at cost 12. Per-user API keys persist server-side in your row — paste them once, use them anywhere.

  • httpOnly sessions

    JWT cookies with Secure + SameSite=Lax in production. 14-day expiry. No fragile localStorage tokens for an attacker to scrape.

  • your keys, your queries

    Etherscan, Helius, CoinGecko, Snusbase, LeakPeek, web3.bio — bring your own. Quotas and audit logs stay on your accounts, not ours.
